3. Package entry point¶

Module entry point.

exception rp2book.R2BError[source]¶

rp2books specific Exception class; used to wrap normal exceptions, without adding any functionality.

rp2book.start(config, package=False, folder=True)[source]¶

Just a shorthand for Book. Using this method ensures the proper setting of the locale to en_US, necessary for the W3C specific date format.

Parameters:
  • config – Reference to the JSON configuration file
  • package – flag on whether the packaged version, ie, the “real” ebook should be generated
  • folder – flag whether the unpacked folder should be kept
